How To Install, How To Set PWA?

First of All, for correct working of any PWA need 4 important files which are:

  • app.js
  • firebase.json
  • manifest.json (Manifest)
  • sw.js (Server Worker)

All this files includes in your template package you need to set only 1 manifest.json file. Please read read below and follow.

Let's dive into the details of how to edit this file!

For Correct setting please open manifest.json file which are in the site-app folder, instead of this text on the 7th line: "Please Copy & Past your PWA URL address here", add your PWA URL address for example like this: "https://yourpwa.com/index.html", very important!! you need https:// (SSL certificate) instead of http://. Without SSL PWA will not works.


Please see screen below


after editing please save manifest.json file, that is all what you need for correct PWA working.
